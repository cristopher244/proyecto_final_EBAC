# Proyecto Final EBAC – Análisis de Datos

## Descripción

Este proyecto corresponde al proyecto final del programa de Ciencia de Datos de EBAC.

Se realiza un análisis exploratorio de datos utilizando Python y librerías de análisis como:

* pandas
* numpy
* matplotlib
* seaborn

## Contenido del proyecto

* Exploración de datos
* 
* Limpieza de datos
* Visualización
* Conclusiones del análisis

## Tecnologías utilizadas

* Python
* Jupyter Notebook
* Pandas
* Matplotlib
* Seaborn
